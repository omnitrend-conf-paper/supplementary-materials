# OmniTrend (Climate Narrative Tracker)

## Structure
* **backend/**: Django API & AI Logic (Python)
* **frontend/**: React Dashboard (JavaScript)

## Quick Start

### 1. Backend Setup
Open a terminal, navigate to the backend, and run the Python environment:
```bash
cd backend
# (Optional) Create venv: python -m venv venv
# (Optional) Activate venv: source venv/bin/activate (Mac) or venv\Scripts\activate (Win)
pip install -r requirements.txt
python main.py  # (Or python manage.py runserver once Django is set up)
